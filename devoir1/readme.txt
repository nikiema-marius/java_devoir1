*****Description*****:
Ce programme permet de gérer les notes des étudiants. 
Il demande le nombre d'étudiants, lit leurs notes, puis calcule et affiche la note du premier et du dernier étudiant, la moyenne de la classe, et le nombre d'étudiants dans différentes catégories (admis, recalés, rattrapage).

*****Comment tester le programme***** :
1. Compiler le programme en utilisant un compilateur Java :
   javac GestionDesNotes.java

2. Exécuter le programme :
   java GestionDesNotes
